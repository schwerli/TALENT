====================================
Acknowledgments
====================================

We would like to express our sincere gratitude to the following repositories and projects for providing helpful components, functions, and ideas that have contributed to the development of TALENT. These projects have significantly supported the integration of various methods into our toolbox:

- `RTDL-revisiting-models <https://github.com/yandex-research/rtdl-revisiting-models>`_ : Provided valuable components for rethinking tabular data models.
- `RTDL-num-embeddings <https://github.com/yandex-research/rtdl-num-embeddings>`_ : Provided key techniques for numerical embeddings in tabular data.
- `Tabular-dl-tabr <https://github.com/yandex-research/tabular-dl-tabr>`_ : Offered insights into deep learning models for tabular data.
- `DANet <https://github.com/WhatAShot/DANet>`_ : Contributed to the development of correlated feature grouping for tabular data.
- `TabCaps <https://github.com/WhatAShot/TabCaps>`_ : Provided the framework for capsule networks used in tabular data tasks.
- `DNNR <https://github.com/younader/dnnr>`_ : Enabled enhanced KNN-based methods for tabular data predictions.
- `PTaRL <https://github.com/HangtingYe/PTaRL>`_ : Supported the creation of a prototype-based regularization framework.
- `Saint <https://github.com/somepago/saint>`_ : Provided row and column attention mechanisms for tabular data modeling.
- `SwitchTab <https://github.com/avivnur/SwitchTab>`_ : Helped in the implementation of self-supervised methods for tabular data.
- `TabNet <https://github.com/dreamquark-ai/tabnet>`_ : Provided attention-based feature selection techniques.
- `TabPFN <https://github.com/automl/TabPFN>`_ : Offered a generalizable pre-trained network for tabular tasks.
- `Tabtransformer-pytorch <https://github.com/lucidrains/tab-transformer-pytorch>`_ : Contributed to transforming categorical features into contextual embeddings.
- `TANGOS <https://github.com/alanjeffares/TANGOS>`_ : Provided key insights into regularization for tabular learning models.
- `GrowNet <https://github.com/sbadirli/GrowNet>`_ : Helped develop gradient boosting frameworks for shallow networks.
- `HyperFast <https://github.com/AI-sandbox/HyperFast>`_ : Supported the use of meta-trained hypernetworks for tabular tasks.
- `BiSHop <https://github.com/MAGICS-LAB/BiSHop>`_ : Provided key elements of sparse Hopfield models for tabular learning.
- `ProtoGate <https://github.com/SilenceX12138/ProtoGate>`_ : Supported the development of prototype-based models for high-dimensional data.
- `Pytabkit <https://github.com/dholzmueller/pytabkit>`_ : Offered helpful tools for efficient tabular data handling.
- `Excelformer <https://github.com/WhatAShot/ExcelFormer>`_ : Contributed techniques for attention-based tabular models.
- `GRANDE <https://github.com/s-marton/GRANDE>`_ : Provided insights for tree-mimic models using gradient descent.
- `AMFormer <https://github.com/aigc-apps/AMFormer>`_ : Supported transformer-based methods for tabular feature interactions.

Additionally, we would like to acknowledge the authors of these works for their invaluable contributions to the field of machine learning on tabular data. Without their innovations, TALENT would not have reached its current state.

If you find any components or resources in TALENT useful in your research, we encourage you to cite these repositories and papers alongside TALENT.
